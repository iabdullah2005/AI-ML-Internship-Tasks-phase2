# AI/ML Advanced Internship Tasks

This repository contains advanced AI and Machine Learning projects completed as part of my internship. Each task demonstrates practical implementation of modern techniques including machine learning pipelines, natural language processing, and conversational AI systems.

---

## 📌 Projects Overview

### 🔹 Task 1: News Topic Classifier (BERT)
- Built a text classification model using a transformer-based architecture
- Fine-tuned DistilBERT on the AG News dataset
- Implemented evaluation using accuracy and F1-score
- Created an interactive app using Gradio

---

### 🔹 Task 2: Customer Churn Prediction Pipeline
- Developed an end-to-end machine learning pipeline
- Performed data preprocessing, feature engineering, and model training
- Used Logistic Regression and Random Forest
- Applied hyperparameter tuning with GridSearchCV

---

### 🔹 Task 4: Context-Aware Chatbot (RAG)
- Built a chatbot using Retrieval-Augmented Generation (RAG)
- Implemented document retrieval using FAISS vector database
- Generated responses using a language model
- Maintained conversational context
