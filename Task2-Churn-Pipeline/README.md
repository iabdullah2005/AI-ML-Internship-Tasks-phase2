# Customer Churn Prediction Pipeline

## 🎯 Objective
Build a production-ready machine learning pipeline to predict customer churn.

## 📂 Dataset
Telco Customer Churn Dataset (Kaggle)

## ⚙️ Approach
- Data preprocessing using Pipeline and ColumnTransformer
- Models used:
  - Logistic Regression
  - Random Forest
- Hyperparameter tuning using GridSearchCV

## 📊 Results
- Achieved good accuracy using Random Forest
- Pipeline ensures reusable and clean workflow

## 💾 Output
- Exported model as: churn_pipeline.pkl

## 🛠️ Tools Used
- Python
- Scikit-learn
- Pandas
- Joblib
